# VybeScript — updates

Public update feed for **VybeScript** (macOS), delivered via [Sparkle](https://sparkle-project.org).

- `appcast.xml` is served over HTTPS by GitHub Pages and is what the app checks for updates.
- Release builds are attached as assets on this repo's **Releases** (they're too large for git).
- VybeScript's source code lives in a separate, private repository.
